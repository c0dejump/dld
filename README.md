# DLD

(Directory Listing Discovery)

Script to check existing files (not jpg, png etc..) in any type of directory listing


## Usage

```
usage: dld.py [-h] [-u URL]

optional arguments:
-h, --help  show this help message and exit
-u URL      URL to scan [required]
```
