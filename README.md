# wud
Files discovery script in wp-upload

Script to check existing files (not jpg, png etc..) in a "wp-upoad


## Usage

```
usage: wud.py [-h] [-u URL]

optional arguments:
-h, --help  show this help message and exit
-u URL      URL to scan [required]
```
